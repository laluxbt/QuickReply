How to use:

Tap any card → message copies to clipboard instantly ✓ badge confirms it
Switch to Discord → paste → send
{username} and {channel} stay as-is for you to replace; {date} and {time} auto-fill on copy
Search by keyword or shortcode like /kyc

To install on phone:

Android Chrome → three dots → Add to Home Screen
iPhone Safari → share icon → Add to Home Screen
